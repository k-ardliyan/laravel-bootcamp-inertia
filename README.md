## Run Locally

Requirements:

- Composer
- Node.js

Clone the project

```bash
  git clone https://github.com/k-ardliyan/laravel-bootcamp-inertia.git
```

Go to the project directory

```bash
  cd laravel-bootcamp-inertia
```

Install dependencies

```bash
  composer dump-autoload
```

Start the server

```bash
  php artisan serve
```

## Tech Stack

**Client:** Inertia, React, TailwindCSS

**Server:** Laravel 11

**Database:** MySQL 8.0
